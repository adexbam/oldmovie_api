Movie api

This is a documentation page for myFlix movie api.

Resource description

Create

POST /users/:username/favorites

POST /users

Read

GET /movies

GET /movies/:title

GET /movies/:title/genre

GET /directors/:name

Update

PUT /users/:username/:password

PUT /users/:username/:email/:dateofbirth

Delete

DELETE /users/:username

DELETE /users/:username/favorites

Example

app.get('/movies', function(req, res) { res.json(movies); });

The above GET request,returns a JSON of a list of ALL movies to the user
