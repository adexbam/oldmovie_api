Movie api

This is a documentation page for myFlix movie api that provides users with access to information about different movies, directors, and genres. Users are able to sign up, update their personal information, and create a list of their favorite movies..


Example

app.get('/movies', function(req, res) { res.json(movies); });

The above GET request,returns a JSON of a list of ALL movies to the user

Method Url Query Request Response
GET /movies localhost:8080/movies Return a list of movies A JSON object holding data about all the movies
GET /movies/:title 8080/movies/:title Return data about a single movie by title A JSON object holding data (description, genre, director, image URL) about a single movie by title to the use
{ title : 'The Godfather', genre : 'Crime', description:'The aging patriarch of an organized crime dynasty transfers control of his clandestine empire to his reluctant son.', director: 'Francis Ford Coppola', imageURL: '#' }
GET /movies/:title/genre 8080/movies/:title/genre Return data about a genre A JSON object holding data for genre (description) by name/title
GET /directors/:name 8080/directors/:name Return info about director by name Return data about a director (bio, birth year, death year) by name
POST /users 8080/users Return users A JSON object holding data about users
POST /users/:username/favorites 8080/users/:username/favorites Allow users to add a movie to their list of favorites A JSON object holding data about users favorite movies
PUT /users/:username/:password 8080/users/:username/:password Allow new users to register Return message
PUT /users/:username/:email/:dateofbirth 8080/users/:username/:email/:dateofbirth Allow users to update their user infor Return message
DELETE /users/:username 8080/users/:username Allow existing users to deregister Return message
DELETE /users/:username/favorites 8080/users/:username/favorites Allow users to remove a movie from their list of favorites Return a message
